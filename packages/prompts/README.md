# ui-pro-prompts

Reusable system prompts from the [UI-Pro](https://github.com/jeanmarcrenaud-prog/UI-Pro)
LangGraph pipeline.  Extracted so they can be versioned, reviewed, and published
independently of the orchestrator code.

## Installation

```bash
pip install ui-pro-prompts
```

## Usage

```python
from ui_pro_prompts import CODING_SYSTEM_PROMPT

print(CODING_SYSTEM_PROMPT)
```

## Contents

| Constant | Node | Language |
|---|---|---|
| `CODING_SYSTEM_PROMPT` | `coding_node` | French |
